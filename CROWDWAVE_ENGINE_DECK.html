Production-Ready Python Engine

Quick Start

# Install pip install crowdwave-engine # Use from crowdwave_engine import CrowdwaveEngine engine = CrowdwaveEngine() config = { "audience": "US consumers 25-54", "geography": "USA", "topic": "Product satisfaction", } questions = [{ "id": "Q1", "text": "How satisfied are you?", "type": "scale", "scale": [1, 5], }] report = engine.simulate(config, questions)
